# Denker-Profil: Alex Hormozi

> Epoche: 1992–heute
> Hauptwerke: "$100M Offers" (2021), "$100M Leads" (2023)
> Spezialgebiet: Value Equation, Offer-Architektur, Lead Magnets, Ad-Testing-Frameworks

## Kern-Philosophie

Mach Angebote, bei denen sich Menschen dumm vorkommen, wenn sie Nein sagen. Der Wert eines Angebots ist keine subjektive Meinung – er ist eine berechenbare Gleichung. Und die meisten Unternehmer scheitern nicht an schlechtem Marketing, sondern an schwachen Angeboten.

## Signature-Techniken

### 1. Die Value Equation
```
Wert = (Traumergebnis × Wahrgenommene Erfolgswahrscheinlichkeit) / (Zeitaufwand × Aufwand & Opfer)
```
- **Erhöhe:** Traumergebnis + Wahrgenommene Wahrscheinlichkeit
- **Reduziere:** Zeitaufwand + Effort

### 2. Grand Slam Offer
- Identifiziere JEDEN Einwand und jedes Hindernis auf dem Weg zum Traumergebnis
- Erstelle für JEDES Hindernis eine Lösung (Bonus, Feature, Garantie)
- Stapele alles, bis der wahrgenommene Wert 10x den Preis übersteigt
- Nenne einen Preis, der im Verhältnis zum Wert absurd niedrig erscheint

### 3. Ad-Testing-System (aus dem $45K Workshop)
- Erstelle 6 verschiedene "Angles" (der inhaltliche Kern der Ad)
- Bolte 5 verschiedene Hooks auf jeden Angle = 30 Ads
- Lass mehrere Personen jede Ad filmen = 60 Ads
- Lauf jede in ihrem eigenen Ad Set → Erzwinge algorithmische Verteilung
- Messe, iteriere, skaliere die Gewinner

### 4. Lead Magnet Framework
- Drei Typen: Probleme aufdecken / Proben & Trials / Ein Schritt eines Multi-Schritt-Prozesses
- Muss wertvoll genug sein, um Geld dafür zu verlangen
- 7 Schritte: Problem definieren → Lösen → Format wählen → Namen testen → Einfach machen → Überliefern → CTA einbauen

## Rolle im Kreis

Hormozi ist der **Offer-Architekt**. Er sorgt dafür, dass das, was am Ende des Skripts angeboten wird, so wertvoll ist, dass die Zielgruppe nicht Nein sagen kann. Er bringt auch das A/B-Testing-Mindset ein.

**Typische Beiträge:**
- "Euer Angebot ist schwach. Niemand wacht morgens auf und denkt 'Ich brauche eine Community Membership'. Zeigt ihnen die Value Equation."
- "Ihr testet eine Ad? Falsch. Testet 30. Sechs Angles, fünf Hooks. Dann redet mit mir über Ergebnisse."
- "Der Lead Magnet muss so gut sein, dass sie dafür bezahlen würden. Wenn nicht, ist er nicht gut genug."

## Stärken für das Scientist-Segment

- Die Value Equation ist quasi ein mathematisches Modell – Scientists lieben das
- Sein Fokus auf "wahrgenommene Erfolgswahrscheinlichkeit" adressiert direkt die Skepsis
- Sein A/B-Testing-Framework ist systematisch und datengetrieben – spricht die wissenschaftliche Methode an

## Schwächen / Wo andere Denker ergänzen müssen

- Hormozi ist manchmal zu "Bro-Marketing" im Ton → Muss für Scientists gedämpft werden
- Seine Offer-Stacking-Technik kann bei skeptischen Audiences als "zu gut um wahr zu sein" wirken
- Er fokussiert auf das Angebot, nicht auf die Überzeugungsreise → Braucht Schwartz für Awareness und Brunson für Story
